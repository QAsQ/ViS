# 可视化伸展树

你可以用ViS来动态观察

* 伸展树旋转的样子


* 删除结点的时候会发生什么



### 一些功能的介绍

在一个节点上左击来选中节点

在空白位置左击来取消节点选中状态

右击唤出菜单

按住空格键并且拖动鼠标左键来拖动画布
